# SkyproWallet

Приложение-кошелек для управления расходами с возможностью авторизации и регистрации.

## Установка приложения:

-   Установить окружение (VS Code, git);
-   Скопировать проект с использованием команды git clone;
-   Установить все зависимости с использованием команды npm install;
-   Запустить приложение, например, в режиме разработки - запустить команду npm run dev.
